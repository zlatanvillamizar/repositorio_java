# 🎵 Conciertos Conectados - Melimarket

Plataforma web para la venta de entradas a conciertos de Melanie Martinez y otros eventos.

## Características

### Administración
- **Login** con credenciales predefinidas (`admin@mail.com` / `123456`)
- **Dashboard** con acceso a módulos:
  - **Categorías**: CRUD completo (crear, editar, eliminar)
  - **Eventos**: CRUD completo con código, nombre, categoría, precio, fecha, hora, ciudad, imagen y descripción
  - **Ventas**: Lista de compras con detalles del cliente y productos

### Clientes
- **Vista principal** con catálogo de eventos
- **Buscador** por nombre de evento
- **Filtros** por ciudad y categoría
- **Detalle de evento** con información completa
- **Carrito de compras** con modal interactivo
- **Formulario de compra** con validación de campos
- **Almacenamiento** de ventas en localStorage

## Tecnologías utilizadas

- HTML5
- CSS3 (con diseño responsivo)
- JavaScript (Vanilla)
- Web Components
- localStorage
- Font Awesome (iconos)
